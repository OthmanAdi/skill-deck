# Code of Conduct

## Our Commitment

We want Skill Deck to be a respectful, inclusive, and constructive open-source project.

## Expected Behavior

- be respectful in discussion and review
- assume good intent, ask for clarification before escalating
- focus critique on ideas and code, not people
- provide actionable feedback

## Unacceptable Behavior

- harassment, discrimination, or personal attacks
- doxxing or sharing private information without consent
- repeated disruptive or hostile behavior

## Enforcement

Project maintainers are responsible for enforcing this code of conduct.

Maintainers may remove comments, close threads, or restrict participation for violations.

## Reporting

If you experience or witness unacceptable behavior, report privately to project maintainers via GitHub private channels.
